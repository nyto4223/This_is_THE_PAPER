考研政治 - 智能复习卡片

这是一个用于考研政治的智能记忆卡片 Web 应用。它基于艾宾浩斯遗忘曲线，使用间隔重复算法 (Spaced Repetition) 帮助用户更科学、更高效地复习知识点。

本项目是一个纯粹的单 HTML 文件，不依赖任何服务器或构建工具，下载后在浏览器中打开即可立即使用。

✨ 主要功能

智能卡池：自动将所有知识点分为“必须复习 (T+1)”、“到期复习 (L2+)”和“新卡片”三个卡池。

间隔重复算法：根据你对卡片“不记得”、“困难”、“记得”的评级，自动计算下一次最佳复习时间 (T+1天, 3天, 7天, 14天...)。

多种学习模式：

常规模式：提供 10 / 20 / 40 张卡片的学习强度，智能混合新卡片和复习卡片。

激进模式：允许你一次性清空所有“必须复习”或“全部到期”的卡片。

进度统计：在“总进度与管理”页面中，可视化展示“未学习”、“易遗忘”和“已掌握”的卡片数量。

数据持久化：所有学习进度自动保存在浏览器的 localStorage 中，关闭浏览器也不会丢失。

数据管理：支持一键“导出学习进度”和“导入学习进度”（JSON 格式），方便你在不同设备间迁移或备份数据。

响应式设计：界面适配桌面和移动设备，随时随地都能复习。

🚀 如何使用

下载本仓库中的 test3-fixed.html 文件。

在你的电脑或手机浏览器中直接打开这个 .html 文件。

开始学习！
